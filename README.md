# Apres Radar

**Don't chase mountains. Chase moments.**

Experience-first mountain discovery platform tracking powder drops, festival vibes, and après scenes across Colorado, Utah, and California.

## 🔥 Features

- Weekend decision engine - "Is this weekend worth it?"
- Real event data from 15+ mountain resorts
- Filter by home city, pass type (Epic/Ikon), and weekend
- Major event detection (festivals, concerts, après scenes)
- Email/SMS alerts for weekly radar drops
